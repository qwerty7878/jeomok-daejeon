# 디자인 — 끄투(KKuTu) 스타일

## 철학
정보 밀도 높게. 여백 적게. 채팅 항상 보이게. 뭔가 계속 움직이게.
모던하고 여백 많은 SaaS UI로 만들면 이 게임의 맛이 죽는다.

## 레이아웃

**PC (≥1024px) — 3분할 고정**
```
[좌: 참가자/목숨]  [중앙: 게임]  [우: 채팅]
```

**모바일 (<1024px) — 하단 탭 3개**
```
[ 게임 ]  [ 채팅 🔴 ]  [ 참가자 ]
```
반응형이 아니라 **두 개의 레이아웃**이다. 분기해서 짜라.

## 토큰 (Tailwind)
- 폰트: Pretendard (한글 필수. @vercel/og에도 임베드)
- 목숨: ♥ (남음) / 💔 (잃는 순간 애니메이션) / 💀 (탈락)
- phase 색: WRITING=파랑, VOTING=보라, RESULT=주황
- 카운트다운 10초 이하 → 빨강 + 펄스

## 절대 원칙
- **P8(ROUND_RESULT) 연출에 시간을 가장 많이 써라.** 여기가 제일 재밌는 15초다.
- 채팅은 절대 접지 마라 (VOTING 잠금 제외).
